# Hands-on Frank-Wolfe at CO@Work

How to run the Pluto notebooks:
```julia
julia> using Pkg
julia> Pkg.activate(".")
julia> Pkg.update()
julia> using Pluto
julia> Pluto.run()
```

This will open your web browser, open the first notebook.

You can also try running the notebooks online on https://binder.plutojl.org (slower).
